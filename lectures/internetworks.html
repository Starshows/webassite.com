Internetworks
1974-1986

Early 1970s

Physical Islands
In addition to ARPANET, other research and education networks were being developed with their own physical links

Early 1970s

Linguistic Islands
Each network could only talk to the other computers on the same network due to proprietary communication protocols

1974

From Islands to Archipelago: Transfer Control Protocol (TCP)
Vint Cerf and Bob Kahn introduced TCP, a communication protocol that replaced ARPANET's NCP, which was more of a device driver for the IMP. Later separated into TCP/IP (Internet Protocol), version 4 became the basic communication language of the Internet.

1974

TCP Agnosticism
TCP is a communication protocol abstracted into four layers, which allows, for example, application developers to assume the bottom three layers are implemented when building apps.

1980

Duke grad students introduce USENET, the first widespread Bulletin Board Service (BBS) that connects one-to-many
Source: Google Groups USENET Archive

1983

Domain Name Service (DNS)
Paul Mockapetris introduces the DNS, a system for converting Uniform Resource Locators (URLs) into IP addresses, replacing the HOSTS.TXT plain-text file previously hosted at Stanford Research Institute (SRI) since the founding of ARPANET

Demo: DNS IP Address Lookup
Type "ping example.com" into a unix terminal window to see the DNS resolve the IP address from the URL

Demo: Full DNS Cycle with traceroute
Type "traceroute example.com" into a unix terminal window to see the route of servers traversed to resolve the IP. Note the trip to *.icann.org, the non-profit LA-based Internet Corporation for Assigned Names and Numbers (ICANN) responsible for all IP designations.

Demo: FTP
Transfer files from client to server using the File Transfer Protocol and an FTP client, such as WinSCP, FileZilla or Transmit

1986
"I asked for a car, I got a computer. How's that for being born under a bad sign?"

Ferris Bueller's Day Off

1986

NSFNET Backbone
The National Foundation for the Sciences (NSF) began to build a 56kbit/s (fast for the time) network linking 6 supercomputers across the US. This network would later be the first Internet "backbone" in the US.

1986

USENET Network
USENET spreads globally as one of the first Bulletin Board Services (BBS), the core application of the fledgling internet next to email

1986

Quantum Link
Quantum Computer Services (later named AOL) introduced Quantum-Link to connect Commodore 64 machines with a dial-up modem

1986

Quantum Link (later AOL) Promo video
Used for connecting Commodore 64 machines with a dial-up modem

Late 1980s

Bulletin Board Service (BBS) Interface
Text-only interface for a BBS in the late 1980s as pioneered by USENET

1981-1991

Exponential Growth
Due to the NSFNET, Q-Link, dial-up modems and the rise of personal computing, the Internet, still in its infancy, begins exponential growth around 1983
